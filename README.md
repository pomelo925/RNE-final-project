# RNE Final Project

## Usage

1. Clone this repo.

```=
cd 
```


2. Run app. (3 terminals)

* pros_app:
  ```
  cd pros_app/ && python3 control.py -s
  ```

* pros_car:
  ```
  cd pros_car/ && ./car_control.sh
  ```

* ros2_yolo_integration:
  ```
  cd ros2_yolo_intergration/ && ros2 run yolo_pkg yolo_detection_node 
  ```
